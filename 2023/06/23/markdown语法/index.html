  • 01 先学会常见的11个

    • 标题

      • 段落前用 # ，后面记得加个空格， # 的个数代表标题的层级数
      • 一二级的标题可以使用两个以上的 = 和 - 来创建

    • 换行

      • 在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行(<br>)。（不过目前对我而言暂无用处）

    • 强调：粗体和斜体

      • 粗体：前后用**，或左右用__(都是双份)
        • logseq里面可直接用插件，选中后用 ctrl+B 加粗
      • 斜体：前后用*，或左右用_(都是单份)
        • logseq里面可直接用插件，选中后用 ctrl+I 转斜体
      • 又粗又斜：前后各添加三个星号或下划线，中间不要带空格。
        • 又粗又斜
        • 又粗又斜

    • 块引用

      • 在段落前添加一个 > 符号

      • 这是一级引用

      • 这是嵌套块引用（在某些系统好像看不出差别）

    • 列表：有序列表和无序列表

      • 有序列表：数字并紧跟一个英文句点，加上空格

      • 无序列表:在每个列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。

    • 代码块

      • 其包裹在反引号 (`) 中
      • ``Use code in your Markdown file.`
      • 多行代码
        • 因为多行代码要控制缩进，可以先输入两行【```】激活代码块，然后直接在代码块中输入
        •   print "hi"

    • 分隔线

      • 在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) 。
      • 注：为了兼容性，请在分隔线的前后均添加空白行。

    • 链接

    • 转义字符

      • 在字符前面添加反斜杠字符 \
      • 特殊字符自动转义： < 和 &
        • 必须要使用实体的形式，像是 < 和 &
        • 在 Markdown 的块级元素和内联元素中， < 和 & 两个符号都会被自动转换成 HTML 实体，这项特性让你可以很容易地用 Markdown 写 HTML。（在 HTML 语法中，你要手动把所有的 < 和 & 都转换为 HTML 实体。

    • 内嵌 HTML 标签（了解即可）

      • 对于 Markdown 涵盖范围之外的标签，都可以直接在文件里面用 HTML 本身。如需使用 HTML，不需要额外标注这是 HTML 或是 Markdown，只需 HTML 标签添加到 Markdown 文本中即可
